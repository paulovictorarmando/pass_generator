# Contributing

Thanks for your interest in contributing!

## Quick start

1. Fork the repo
2. Create a branch for your change
3. Make your change with a clear commit message
4. Open a Pull Request

## What to work on

Good contributions include:

- Bug fixes and small correctness improvements
- Improving the CLI experience (input validation, clearer messages)
- Documentation improvements
- Adding lightweight tests (if/when a test setup is introduced)

## Development notes

- This project uses only the Python standard library.
- Run locally with:

```bash
python3 main.py
```

When filing an issue, please include:

- OS and Python version
- Steps to reproduce
- Expected vs actual output
